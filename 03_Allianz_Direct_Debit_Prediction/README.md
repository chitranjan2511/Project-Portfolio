# Customer Payment Method Prediction using Machine Learning

## Project Overview

This project uses machine learning to predict which insurance customers are most likely to adopt direct debit as their payment method. The objective is to help financial institutions improve cash flow management and reduce administrative costs.

## Business Problem

Only 21% of Allianz insurance contracts use direct debit, while the industry average is around 65%. This project develops a predictive model to identify customers most likely to adopt direct debit.

## Methodology

The project follows the CRISP-DM data science framework.

Key steps:

* Data import and preprocessing using Python
* Exploratory data analysis
* Dashboard analysis using Power BI
* Dataset partitioning (70% training, 30% validation)
* Model building using Random Forest classifier
* Model evaluation using multiple metrics

## Model Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix
