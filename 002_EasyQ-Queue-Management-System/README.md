# EasyQ – Digital Queue Management System

🚀 Skip the Line, Save the Time

## 📌 Overview
EasyQ is a digital queue management system that reduces waiting time in clinics using real-time tracking and smart appointment scheduling.

## 🎯 Problem
Patients waste ~2 hours per visit due to:
- No queue visibility
- Uncertain doctor availability
- Overcrowding

## 💡 Solution
- Real-time queue tracking
- Appointment booking
- Notifications & alerts
- Remote waiting (wait from home)
- Clinic analytics

## 📊 Business Insight
- TAM: ₹75 Cr
- SAM: ₹30 Cr
- SOM: ₹1.5–3 Cr
