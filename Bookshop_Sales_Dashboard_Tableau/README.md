# Bookshop Sales Dashboard (Tableau)

## Project Overview

This project presents an interactive Tableau dashboard developed to analyze bookshop sales data. The dashboard visualizes sales trends, product performance, and category-wise revenue to support data-driven business decisions.

## Objective

To analyze book sales data and identify key patterns in product demand, revenue generation, and sales distribution across different book categories.

## Dataset

The dataset includes information related to:

* Book categories
* Sales values
* Order details
* Revenue performance

## Key Dashboard Insights

* Overall sales performance
* Category-wise revenue distribution
* Top-selling books
* Sales comparison across different categories
* Identification of high-demand book genres

## Tools Used

Tableau
Microsoft Excel

## Skills Demonstrated

Data visualization
Dashboard development
Sales trend analysis
Business intelligence reporting
