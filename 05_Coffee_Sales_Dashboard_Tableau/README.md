# Coffee Sales Dashboard (Tableau)

## Project Overview

This project presents an interactive sales dashboard built using Tableau to analyze coffee product sales and office supply sales. The dashboard helps visualize sales trends, product performance, and category-wise revenue distribution.

## Objective

To analyze sales performance and identify key business insights that can help improve decision-making in retail operations.

## Dataset

The dataset contains sales records including:

* Product categories
* Sales amount
* Order details
* Customer purchasing patterns

## Key Dashboard Insights

* Overall sales trends across time
* Best-performing coffee products
* Category-wise revenue distribution
* Comparative performance of coffee vs office supplies
* Identification of high-performing products

## Tools Used

Tableau
Microsoft Excel

## Skills Demonstrated

Data cleaning
Data visualization
Interactive dashboard design
Business insights generation
