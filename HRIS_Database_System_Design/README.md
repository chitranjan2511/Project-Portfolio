# Human Resource Information System (HRIS) Database Design

## Project Overview

This project designs a Human Resource Information System (HRIS) for HCL Tech to manage employee data, payroll, attendance, performance evaluation, and compliance reporting.

## Objective

To develop a centralized HR database system that automates HR operations and supports data-driven workforce management.

## Key System Components

Employee Management
Stores employee personal details, job roles, departments, and employment history.

Attendance Tracking
Records employee attendance, work hours, and overtime details.

Payroll Management
Manages salary components including allowances, deductions, and net salary calculations.

Leave Management
Tracks employee leave requests, approvals, and leave balances.

Performance Evaluation
Stores employee performance scores, feedback, and training recommendations.

## Database Tables

Employee Table
Attendance Table
Payroll Table
Leave Table
Performance Evaluation Table

## Sample Queries

Department-wise employee count
Employee attendance summary
Employees on leave during a specific period
