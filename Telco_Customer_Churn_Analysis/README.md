# Telco Customer Churn Analysis

## Project Overview

This project analyzes customer churn in the telecommunications industry using Python. The objective is to identify patterns in customer behavior that lead to churn and help businesses improve customer retention strategies.

## Objective

To analyze telecom customer data and identify key factors influencing customer churn.

## Dataset

The dataset contains information about telecom customers including:

* Customer demographics
* Service subscriptions
* Contract types
* Monthly charges
* Total charges
* Churn status

## Key Analysis Performed

* Data cleaning and preprocessing
* Exploratory data analysis
* Customer churn distribution analysis
* Feature relationship analysis
* Identification of factors influencing churn

## Tools Used

Python
Pandas
NumPy
Matplotlib / Seaborn

## Skills Demonstrated

Data analysis
Exploratory data analysis (EDA)
Business insights generation
Customer behavior analysis
